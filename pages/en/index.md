# Introduction

IMKIT Platform API is the core API service of IMKIT Chat Server, developed by [FUNTEK](https://funtek.co/), designed specifically for developers integrating the IMKIT SDK. It enables you to quickly build real-time messaging services for various use cases. Whether it's 1-on-1 private conversations or multi-user group chats, IMKIT Platform API can meet your needs.

You can also visit our [official website](https://imkit.io/) to learn more about IMKIT products.

## Advantages of IMKIT Platform API

### Rapid Integration

Compared to traditional chat feature development, IMKIT Platform API allows you to set up a complete real-time messaging backend service within hours, eliminating the need to spend significant time building chat servers, so you can focus on developing your core product.

- Ready-to-use chat API service upon registration
- Comprehensive API support covering users, chat rooms, and real-time messaging
- Flexible scenario support capabilities
- Automatic message routing and storage

### Comprehensive API Features

IMKIT Platform API provides the API features needed to build most chat applications.

- **User Management API** - Create and update user profiles
- **Chat Room Management API** - Create group and one-on-one chat rooms
- **Messaging API** - Support for text, images, files, and other multimedia messages
- **Administration API** - Sensitive word filtering, chat room member management, user blocking

### Enterprise-Grade Reliability

Built on FUNTEK's years of experience serving large-scale projects and cloud infrastructure, providing highly available API services that support large-scale concurrent request processing.

### Security Assurance

- Dual authentication with API Key and JWT Token
- End-to-end message encryption
- Complete API call logging and tracking
