# はじめに

IMKIT Platform API は、[FUNTEK](https://funtek.co/) が開発した IMKIT Chat Server のコア API サービスです。IMKIT SDK を統合する開発者向けに設計されており、さまざまなユースケースにおけるリアルタイムメッセージングサービスを迅速に構築できます。1対1のプライベート会話から複数ユーザーのグループチャットまで、IMKIT Platform API があらゆるニーズに対応します。

[公式ウェブサイト](https://imkit.io/) でも IMKIT 製品に関する詳細情報をご覧いただけます。

## IMKIT Platform API の利点

### 迅速な統合

従来のチャット機能開発と比較して、IMKIT Platform API を使用すると数時間以内に完全なリアルタイムメッセージングバックエンドサービスをセットアップできます。チャットサーバーの構築に多大な時間を費やす必要がなくなり、コアプロダクトの開発に集中できます。

- 登録後すぐに使えるチャット API サービス
- ユーザー、チャットルーム、リアルタイムメッセージングをカバーする包括的な API サポート
- 柔軟なシナリオ対応能力
- 自動メッセージルーティングとストレージ

### 充実した API 機能

IMKIT Platform API は、ほとんどのチャットアプリケーションを構築するために必要な API 機能を提供します。

- **ユーザー管理 API** - ユーザープロファイルの作成・更新
- **チャットルーム管理 API** - グループおよび1対1チャットルームの作成
- **メッセージング API** - テキスト、画像、ファイル、その他のマルチメディアメッセージのサポート
- **管理者 API** - 不適切ワードフィルタリング、チャットルームメンバー管理、ユーザーブロック

### エンタープライズグレードの信頼性

FUNTEK が長年にわたって大規模プロジェクトとクラウドインフラを提供してきた経験に基づき、大規模な同時リクエスト処理をサポートする高可用性 API サービスを提供します。

### セキュリティ保証

- API Key と JWT Token によるデュアル認証
- エンドツーエンドのメッセージ暗号化
- 完全な API 呼び出しのロギングと追跡
