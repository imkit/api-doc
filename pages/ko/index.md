# 소개

IMKIT Platform API는 [FUNTEK](https://funtek.co/)이 개발한 IMKIT Chat Server의 핵심 API 서비스입니다. IMKIT SDK를 연동하는 개발자를 위해 설계되었으며, 다양한 사용 사례에 맞는 실시간 메시징 서비스를 빠르게 구축할 수 있습니다. 1:1 개인 대화부터 다중 사용자 그룹 채팅까지, IMKIT Platform API가 모든 요구를 충족시킵니다.

[공식 웹사이트](https://imkit.io/)에서 IMKIT 제품에 관한 자세한 정보를 확인하실 수 있습니다.

## IMKIT Platform API의 장점

### 빠른 연동

기존 채팅 기능 개발과 비교하여, IMKIT Platform API를 사용하면 몇 시간 내에 완전한 실시간 메시징 백엔드 서비스를 구축할 수 있습니다. 채팅 서버 구축에 많은 시간을 할애할 필요 없이 핵심 제품 개발에 집중할 수 있습니다.

- 가입 즉시 사용 가능한 채팅 API 서비스
- 사용자, 채팅방, 실시간 메시징을 아우르는 포괄적인 API 지원
- 유연한 시나리오 대응 능력
- 자동 메시지 라우팅 및 저장

### 풍부한 API 기능

IMKIT Platform API는 대부분의 채팅 애플리케이션 구축에 필요한 API 기능을 제공합니다.

- **사용자 관리 API** - 사용자 프로필 생성 및 업데이트
- **채팅방 관리 API** - 그룹 및 1:1 채팅방 생성
- **메시징 API** - 텍스트, 이미지, 파일 및 기타 멀티미디어 메시지 지원
- **관리자 API** - 금칙어 필터링, 채팅방 멤버 관리, 사용자 차단

### 엔터프라이즈급 신뢰성

FUNTEK이 오랜 기간 대규모 프로젝트와 클라우드 인프라를 운영한 경험을 바탕으로, 대규모 동시 요청 처리를 지원하는 고가용성 API 서비스를 제공합니다.

### 보안 보장

- API Key와 JWT Token을 이용한 이중 인증
- 엔드투엔드 메시지 암호화
- 완전한 API 호출 로깅 및 추적
